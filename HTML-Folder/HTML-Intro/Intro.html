What is html?

HTML is a markup language used in building or developing the structure of an app {either web or mobile}. full meaning of HTML is:
   - Hyper Text:HTML uses the HTTP part of the internet
   - markup: the written code is annoted with keywords
   - Language: HTML can be read by both human and computer

What is HTML5?
